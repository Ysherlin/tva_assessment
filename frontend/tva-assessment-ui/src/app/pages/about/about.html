About This Application

This application was built as part of a technical assessment to demonstrate full-stack development skills using modern technologies and best practices.

Purpose

The goal of the application is to manage persons, their accounts, and related transactions, while enforcing realistic business rules such as account status and transaction validation.

Technology Stack

  • Backend: .NET 8 Web API
  • Frontend: Angular (Standalone Components)
  • ORM: Entity Framework Core
  • Database: SQL Server

Design Approach

The solution follows clean separation of concerns, with clear boundaries between controllers, services, repositories, and domain entities. Business rules are enforced in the application layer, and the frontend mirrors those rules to provide a consistent user experience.
